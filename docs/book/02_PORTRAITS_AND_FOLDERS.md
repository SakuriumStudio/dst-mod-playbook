# 02 portraits：参照フォルダ地図

## 代表フォルダと用途
- igportraits/：大判ポートレート
- images/selectscreen_portraits/：キャラ選択画面
- images/saveslot_portraits/：セーブスロット
- images/avatars/：小アイコン（プロフィール等）
- images/names/ / images/names_gold/：名前プレート系（ズレやすい）

重要：**用途が違う＝探されるキー（region）も違う**。
